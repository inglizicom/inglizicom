# Picture filenames — all 20 units

Drop JPG / PNG / WebP files into the matching `unit0X/` folder. Square 800×800 works best, but any aspect ratio works — the card crops to fit.

The renderer prefers local files; if a local file is missing, it falls back to a generated photo (Pollinations) or a designed placeholder. So you can roll out images one unit at a time.

> Only **Unit 1** is currently wired to look at local paths in code (`src/data/private-lessons/unit01.ts`). When you finish a unit's images, ping me and I'll wire it the same way.

> warm friendly mood, real-life scene at home or outdoors, square 1:1,
> consistent style with previous images in the set.
