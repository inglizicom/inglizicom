UNIT 8  —  pictures folder
========================

Drop your pictures for this unit here, named by phrase ORDER:

  h1.jpg   = phrase 1 of this unit
  h2.jpg   = phrase 2
  h3.jpg   = phrase 3   ... and so on

(.png / .webp also work.) The letter "h" = unit 8. Number = the
vocabulary phrase position as it appears in the deck. Landscape
images (e.g. 1200x900) look best. Reload the deck after adding —
no restart needed.
