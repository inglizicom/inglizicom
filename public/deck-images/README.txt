YOUR OWN DECK IMAGES
====================

Put your pictures here to use them in the teaching decks instead of the
automatic Unsplash photos. Your image always wins — Unsplash is only used for
phrases you haven't added a picture for. There are TWO ways to name them.


WAY 1 (recommended): BY UNIT FOLDER + ORDER
-------------------------------------------
Each unit has its own folder: unit-1, unit-2, … unit-20.
Inside a folder, name the file by the phrase's ORDER, with a letter for the unit:

  unit-1/a1.jpg   = phrase 1 of Unit 1      (unit 1 = "a")
  unit-1/a2.jpg   = phrase 2 of Unit 1
  unit-2/b1.jpg   = phrase 1 of Unit 2      (unit 2 = "b")
  unit-2/b2.jpg   = phrase 2 of Unit 2
  ...

Letter = unit (a=1, b=2, c=3 …). Number = the vocabulary phrase position as it
appears in the deck. Each folder has a README reminding you of its letter.

Note: this matches by position, so if you reorder/add phrases in a unit the
pictures shift — just renumber the files to match.


WAY 2: BY PHRASE NAME (in this top folder)
------------------------------------------
Name the file after the English phrase: lowercase, spaces/punctuation become
hyphens "-", ending in .jpg (or .png / .webp):

  "I take a shower"      ->  i-take-a-shower.jpg
  "I brush my teeth"     ->  i-brush-my-teeth.jpg

Open the deck index (/admin/present) and click "Download the filename guide"
for the exact list of every phrase.


NOTES
-----
- A positional file (Way 1) is checked first; then the phrase-name file (Way 2);
  then the automatic photo.
- Landscape images look best (e.g. 1200x900).
- After adding files, just reload the deck — no restart needed.
- These files are committed with the project, so they also work in production
  and inside the offline export.
