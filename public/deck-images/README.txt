YOUR OWN DECK IMAGES
====================

Put your pictures in THIS folder to use them in the teaching decks instead of
the automatic Unsplash photos. Your image always wins — Unsplash is only used
for phrases you haven't added a picture for.

HOW TO NAME EACH FILE
---------------------
Name the file after the English phrase on the slide:
  - all lowercase
  - spaces and punctuation become hyphens "-"
  - end with .jpg (or .png / .webp)

Examples:
  "I take a shower"      ->  i-take-a-shower.jpg
  "I brush my teeth"     ->  i-brush-my-teeth.jpg
  "How much is the ticket?" -> how-much-is-the-ticket.jpg

Need the exact list for every phrase? Open the deck index
(/admin/present) and click "Image filename guide" to download the full list,
grouped by unit.

NOTES
-----
- Landscape images look best (e.g. 1200x900).
- After adding files, just reload the deck — no restart needed.
- These files are committed with the project, so they also work in production
  and inside the offline export.
